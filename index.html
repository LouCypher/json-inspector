Overview

Inspect JSON object from a URL or entered data.
Requires either DOM Inspector or DOM Inspector Plus! [dm] extension.

Compatibility

  • Firefox 17.0 - 26.*
  • SeaMonkey 2.14 - 2.23.*
  • Thunderbird 17.0 - 26.*
